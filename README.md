# ldgni.io

> My portfolio

## Running Locally

1. Clone the repository

```sh
git clone https://github.com/ldgni/ldgni.io.git
```

2. Navigate inside the repository

```sh
cd ldgni.io
```

3. Install dependencies

```sh
pnpm install
```

4.  Start the development server

```sh
pnpm dev
```
